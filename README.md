# sweep-mine
### 原生js开发 原型编程 构建工具压缩代码 对象存储扫雷的数据以及标记等行为
>原生js开发，实现双击了，单击，高度复原windows上的扫雷游戏，并在性能上做了相应的优化。
>采用原型上编程，实现的局部功能的分离，保证的相应独立功能模块的独立完整，尽量实现的了模块的功能不收其他的功能模块的影响
## 设计：仿照windows 系统的扫雷的界面设计，并在适合网页上做了优化和处理，使得加载速度更快了，游戏更流程
